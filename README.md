### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/YoussefSalem582/flutter_architecture_patterns.git
   ```

   ```bash
   cd flutter_architecture_patterns/mvvm_architeture_pattern
   ```

   ```bash
   flutter pub get
   ```

4. Run the project:
   ```bash
   flutter run
   ```
